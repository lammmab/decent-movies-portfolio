# The decentralized backend for decent-movies-portfolio

## If you haven't, check out [the main repo](https://github.com/lammmab/decent-movies-portfolio)

### Requirements:
node.js
npm

### Usage instructions:
1. Run to install dependencies 
```bash
npm install
```
2. Create a `.env` file in this directory
3. Inside the .env, add `KEY="(asecretkey)"`
4. Head into `config.json`
5. Tweak parameters to your liking
6. Depending on your use case, run either:
```bash
npm run build   # To build the project
npm run dev     # To run in development mode
```
